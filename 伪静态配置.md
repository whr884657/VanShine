# VanShine 伪静态 / URL 重写配置

分享链接标准格式：**`{域名}/d/index.php?token={32位token}`**

预览/下载：**`/d/index.php?token=...&stream=1&file=ID`**

---

## 重要：防止 PHP 源码被下载

若访问 `https://域名/d/index.php` 时浏览器**直接下载** `.php` 文件，说明 Nginx 把 PHP 当成了静态文件，**必须**为 `/d/` 下的 PHP 配置解释器（FastCGI），不能只靠 `try_files $uri`。

---

## Nginx（推荐）

在 `server { ... }` 内添加（PHP 配置块请替换为你面板实际路径，如宝塔 `enable-php-82.conf`）：

```nginx
# 1. /d 目录下 PHP 必须走解释器（防止 index.php 被当作静态文件下载）
location ~ ^/d/.+\.php$ {
    include enable-php-82.conf;
}

# 2. 旧版 /d/{token} 自动跳转到标准 query 链接（可选）
location ~ ^/d/([A-Za-z0-9]{16,64})$ {
    return 301 /d/index.php?token=$1;
}

# 3. 站点通用伪静态
location / {
    try_files $uri $uri/ $uri.php$is_args$args;
}
```

### 说明

| 访问路径 | 处理 |
|----------|------|
| `/d/index.php` | 由 PHP 解释器执行 → 无 token 时显示安全提醒 |
| `/d/index.php?token=xxx` | 公开分享页 |
| `/d/index.php?token=xxx&stream=1&file=ID` | 文件流下载/预览 |
| `/d/旧token` | 301 跳转到带 `?token=` 的标准链接 |

---

## Apache

根目录 `.htaccess` 与 `d/.htaccess` 已配置；需启用 `mod_rewrite` 且 `AllowOverride All`。Apache 环境下 `/d/index.php` 通常不会触发源码下载。

---

## 验证

1. 访问 `https://你的域名/d/index.php` → 应显示 HTML 安全提醒页（**不是**下载文件）
2. 创建分享后打开 `https://你的域名/d/index.php?token=...` → 应显示分享页
3. 下载按钮链接含 `stream=1&file=` 参数，可正常下载

## 常见问题

**Q：为什么 `try_files`  alone 会导致下载 PHP？**  
`try_files $uri` 命中磁盘上的 `d/index.php` 时，若未匹配到 PHP-FPM，Nginx 会按静态文件返回源码。

**Q：旧链接 `/d/{token}` 还能用吗？**  
可以，会自动 301 重定向到 `/d/index.php?token={token}`。
