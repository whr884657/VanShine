# VanShine 伪静态 / URL 重写配置

分享功能**不需要**任何额外 Nginx 规则，与全站共用同一套通用伪静态即可。

---

## 分享链接格式

| 用途 | 地址 |
|------|------|
| 安全提醒 | `{域名}/d/` 或 `{域名}/d/index.php` |
| 公开分享页 | `{域名}/d/?token={32位token}` |
| 下载/预览 | `{域名}/d/?token=...&stream=1&file=ID` |

URL 中**不出现** `index.php` 路径段；由目录默认首页 `d/index.php` 处理请求。

旧版 `/d/index.php?token=`、`/d/index.php/{token}` 会自动 301 跳转到 `/d/?token=`。

---

## Nginx / Apache

只需站点**通用**伪静态：

```nginx
location / {
    try_files $uri $uri/ $uri.php$is_args$args;
}
```

- **不要**为分享单独添加 `/d/` 规则
- **不要**在文档或配置里写死 PHP 版本号（如 `enable-php-82.conf`）
- PHP 解释由面板/环境自带的 `.php` 处理配置负责

若曾添加 `location ^~ /d/ { try_files $uri ... }` 导致 `index.php` 源码被下载，请删除后仅保留上面通用规则。

Apache 用户：根目录 `.htaccess` 仅保护 `config/`；`d/.htaccess` 启用 `AcceptPathInfo`（兼容旧 PATH 链接跳转）。

---

## 验证

1. `https://你的域名/d/` → 安全提醒页 + 底部版权/备案
2. `https://你的域名/d/?token=...` → 分享页
3. 不应出现下载 `.php` 文件
