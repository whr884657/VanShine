# VanShine 1.0.15 发行说明

**发行日期：** 2026-06-26  
**压缩包名称：** `VanShine1.0.15.zip`  
**适用环境：** PHP 7.4+ / MySQL 5.7+ / 需 ZipArchive 扩展（在线更新功能）

---

## 版本概述

本版本在 v1.0.13 基础上，新增 **Gitee 在线更新**、**数据库增量迁移**、**统一弹窗组件** 等能力，并修复退出登录、浏览器标题重复等问题，同时加强数据库配置文件保护与文档脱敏。

---

## 详细更新内容

### 一、在线更新（v1.0.14）

- 登录后台后**自动检测** Gitee 仓库最新版本，无需手动检查
- 发现新版本时使用**统一 PHP 弹窗**展示版本号与更新说明列表
- 支持**一键下载** Gitee 发行包、自动覆盖安装、完成后**自动清理**临时文件
- 本地版本**高于**仓库版本时不提示（便于开发测试环境）
- 新增 `update.json` 作为远程版本清单，发版时需与 `core/version.php` 同步

### 二、数据库迁移（v1.0.15）

- 在线更新完成后，自动执行 `install/migrations/` 目录下**尚未应用**的 SQL 脚本
- 已执行版本记录在数据库配置表 `schema_migrations` 中，避免重复执行
- 本版附带迁移 `1.0.15.sql`：为已安装站点补全 `site_logo` 配置项（若缺失）

### 三、配置安全加固（v1.0.15）

- **`config/database.php` 绝不被覆盖**：列入硬编码保护名单
- 更新前记录文件指纹，更新后再次校验，若被意外修改则**中止并报错**
- `config/install.lock` 同样不会被更新包替换

### 四、界面与体验优化（v1.0.12 ~ v1.0.13）

- 修复点击「退出」无法登出的问题
- 侧边栏底部、顶部栏增加「退出登录」入口，手机端顶部栏可退出
- 电脑端侧边栏展开时隐藏顶部栏重复站点名称
- 浏览器标题去重（修复「站点名 - 站点名」问题）
- 系统设置改为**折叠板块**，绑定子域名列表改为**卡片展示**（手机端无需横滑）
- 全站统一自定义弹窗，替代浏览器原生 `alert` / `confirm`
- 安装向导数据库默认地址改为 `localhost`

### 五、安全与文档（v1.0.15）

- 移除 README、安装页、前台页中对**密码加密算法**的公开描述
- 降低开源仓库被针对性爆破的风险

## 安装与升级

### 全新安装

1. 下载 `VanShine1.0.15.zip` 并解压到 Web 目录  
2. 确保 `config/`、`storage/` 目录可写  
3. 创建 MySQL 空数据库  
4. 浏览器访问 `https://你的域名/install` 完成五步安装  

### 从旧版升级

**方式一（推荐）：** 登录后台，若检测到新版本，点击弹窗「立即更新」自动完成。  

**方式二（手动）：** 下载本发行包解压覆盖（**不要覆盖** `config/database.php` 和 `config/install.lock`），然后访问后台触发迁移或重新登录一次。
