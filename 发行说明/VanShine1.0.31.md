# VanShine 1.0.31 发行说明

**发行日期：** 2026-06-26  
**压缩包名称：** `VanShine1.0.31.zip`

---

## 版本概述

修正储存 KEY 编号为连续 1–7，完善文件管理交互，并更新 README 目录结构说明。

---

## 主要变更

### 储存 KEY 调整

| KEY | 储存 |
|-----|------|
| 1 | 本地 |
| 2 | AWS S3 |
| 3 | 阿里云 OSS |
| 4 | 腾讯云 COS |
| 5 | 七牛云 Kodo |
| 6 | 蓝奏云优享版 |
| 7 | WebDAV（原 KEY 9） |

升级后自动将已绑定 WebDAV 的文件夹/文件记录从 KEY 9 迁移为 7。

### 文件管理增强

- 批量上传（多选文件）
- 拖拽上传到当前文件夹
- 文件夹重命名
- 储存连接测试（系统设置 → 各储存子板块）

### 文档

- 根目录 `README.md` 目录结构已与当前仓库对齐

---

## 升级说明

- **有数据库结构变更**：执行 `1.0.31.sql`（KEY 迁移）
- 建议在升级前备份数据库与 `upload/`、`data/` 目录
