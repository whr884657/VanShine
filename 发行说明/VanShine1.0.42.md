# VanShine 1.0.42 发行说明

**发行日期：** 2026-06-27  
**压缩包名称：** `VanShine1.0.42.zip`

---

## 版本概述

修复文件管理删除不刷新、上传误报失败、链接显示编码等问题；优化预览弹窗与全局提示样式。

---

## 主要变更

### 文件管理

- **删除即时刷新**：删除文件后列表立即更新，无需手动刷新页面
- **预览联动**：在预览弹窗中删除文件时自动关闭弹窗
- **链接可读化**：预览框中的分享链接显示解码后的中文路径（如 `/upload/本地/6.png`）
- **弹窗优化**：电脑端预览弹窗加宽（最大约 960px），标题显示 **存储名** 而非原始文件名

### 上传修复

- 文件管理页上传改为当前页面直接 XHR，避免桥接窗口导致的 JSON 解析失败
- 增强 AJAX 响应解析（去除 BOM、提取 JSON 对象）
- 服务端输出 JSON 前清理输出缓冲，减少杂讯干扰

### 全局提示

- 新增 **胶囊 Toast** 组件（`VsToast`）
- 文件管理、系统设置等页面的保存、复制、上传成功提示改为顶部弹出胶囊，替代绿色长条

---

## 升级说明

- **无数据库结构变更**
- 升级后 **Ctrl+F5** 强刷后台以加载新 JS/CSS
