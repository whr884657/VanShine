# VanShine 1.0.60 发行说明

**发行日期：** 2026-06-27  
**压缩包名称：** `VanShine1.0.60.zip`

---

## 变更内容

### 分享页 UI 重设计

- 统一 **卡片式页面外壳**（顶栏站点名 + 标题区 + 内容 + 页脚），消除原先顶部大片空白、底部备案「临时贴上去」的割裂感
- 分享页、密码验证页、安全提醒页（`/d/` 无 token）视觉风格一致
- 备案/版权信息整合进页脚条，与法律提示同一区域展示

### 文件展示优化

- 文件列表增加 **类型图标**（图片、视频、压缩包、PDF、文档、代码等）
- 开启预览时，**图片文件显示缩略图**
- **单文件分享**默认展开在线预览区（图片等内容直接可见，不再只有下方列表）
- 文件夹分享：桌面端预览与文件列表并排，移动端点击文件后自动滚到预览区

---

## 升级说明

- **无数据库变更**，覆盖文件即可（保留 `config/`）
