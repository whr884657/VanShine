# VanShine 1.0.65 发行说明

**发行日期：** 2026-06-27  
**压缩包名称：** `VanShine1.0.65.zip`

---

## 变更内容

### 公开分享页

- **取消自动预览**：打开分享链接后不再自动加载第一个文件
- **点击后预览**：访客点击列表中的文件名后，才加载对应文件预览
- **动态预览框**：文件夹分享初始仅显示文件列表；点击文件后预览区出现并进入双栏/上下布局
- 单文件分享同样需点击文件名后才会显示预览区

### 分享管理

- 复制短链接：支持剪贴板 API 失败时的 `execCommand` 回退
- 复制成功时顶部闪动提示「链接已复制到剪贴板」，按钮短暂显示「已复制」

---

## 升级说明

- **无数据库变更**，覆盖文件即可（保留 `config/`）
