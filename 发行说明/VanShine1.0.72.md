# VanShine 1.0.72 发行说明

**发行日期：** 2026-06-27  
**压缩包名称：** `VanShine1.0.72.zip`

---

## 变更内容

### EdgeOne API 修复

- **DescribeContentQuota**：补充必填 `ZoneId`，修复配额信息不可用
- 移除 **EdgeKV** 展示（账户未开通 KV 存储时不再报错）
- 预热/刷新记录为空时显示「暂无记录」而非 `[]`

### EdgeOne 后台扩展

页内分组导航由 8 项扩展为 **18 个功能页**：

| 分组 | 页面 |
|------|------|
| 总览 | 概览 |
| 站点与域名 | 站点管理、域名加速、七层加速、别称域名、DNS 记录 |
| 内容与函数 | 内容管理、边缘函数 |
| 安全与证书 | 安全策略、证书管理、源站防护 |
| 网络与负载 | 四层代理、负载均衡 |
| 监控与计费 | 数据分析、日志服务、套餐计费 |
| 配置与扩展 | 配置版本、扩展功能 |

各页面按站点 ZoneId 调用对应 API，单接口失败不影响其他模块。

---

## 升级说明

- **无数据库变更**：覆盖文件即可，保留 `config/database.php`
- 旧链接 `/admin/cdn/edgeone/monitor.php` 自动跳转至套餐计费页
