# VanShine 1.0.75 发行说明

## 变更内容

### EdgeOne 后台 UI 可视化重构

- 新增 `data-view` 统一渲染层，将 API 原始 JSON 转为可读界面
- **内容管理配额**：卡片 + 日配额进度条（刷新 / 预热分类）
- **数据分析**：七层 / 四层 / 回源时序以表格展示，流量自动格式化
- **站点验证 / CNAME**：验证状态 badge、文件路径与 NS 列表卡片化
- **各模块列表数据**：加速域名、DNS、函数、证书等自动推断表格列
- **概览页**：当前站点卡片高亮；各 section 可折叠查看原始 JSON
