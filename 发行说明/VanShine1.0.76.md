# VanShine 1.0.76 发行说明

**发行日期：** 2026-06-28  
**压缩包名称：** `VanShine1.0.76.zip`

---

## 变更内容

### EdgeOne 数据分析（完整查询能力）

- 支持 **七层访问 / 回源 / 四层** 三类数据源切换
- 指标：响应流量、客户端请求流量、访问请求数、带宽、平均耗时、缓存命中等
- 时间范围：近 2 小时 / 24 小时 / 7 天 / 30 天
- 时间粒度：1 分钟 / 5 分钟 / 1 小时 / 1 天
- 展示：KPI 合计/平均/峰值 + **折线图** + 明细表

### EdgeOne 套餐计费（业务视图）

- 套餐卡片：名称、编号、状态、生效/到期、服务区域、配额
- 绑定站点列表（域名 + ZoneId）
- 站点用量：今日加速流量、缓存命中、安全请求、本月加速流量
- 计费用量趋势：可选计费指标 + 时间范围/粒度 + 折线图

### UI

- **已移除**「查看原始 JSON」及一切 raw API 数据直出
