# VanShine 1.0.77 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.0.77.zip`

---

## 变更内容

### EdgeOne 概览仪表盘

- 首页改为**多指标统计图墙**：响应/请求流量、带宽、访问次数、响应耗时等统一展示
- **统一筛选**：时间范围（1 分钟~31 天）、站点（全部/单个）、域名（可选）一次配置，全部图表同步更新
- 全部站点默认显示**分站点曲线 + 总计曲线**

### 导航与交互

- **电脑端**：顶部横向大类导航，多子项大类悬停下拉
- **手机端**：分组折叠导航，仅展开当前相关分组
- EdgeOne 内页切换、查询、切换站点改为**局部刷新**，不再整页 reload

### 站点状态与配额

- 修正 CNAME 接入站点误显示「待配置」：结合加速状态综合判断为「运行中」
- 概览「套餐与配额」：展示加速流量、安全请求、四层流量配额及今日用量；保留刷新/预热配额

- **无数据库变更**：覆盖文件即可
