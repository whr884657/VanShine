# VanShine 1.0.82 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.0.82.zip`

## 变更内容

### EdgeOne 手机端导航 UX

- 由多个并列大折叠改为 **一个总折叠抽屉**
- **默认收起**：仅显示一条（当前所在「大类 · 页面」）
- 点击展开后，各大类可再次折叠查看子项
- 点击子项跳转后，总折叠 **自动收起**，恢复为一条

电脑端横向导航不变。
