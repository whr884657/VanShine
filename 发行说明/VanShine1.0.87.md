# VanShine 1.0.87 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.0.87.zip`  
**版本类型：** 小版本（EdgeOne 概览体验优化）

---

## 变更内容

### 时间范围精简

可选范围调整为：**近 30 分钟、1 小时、6 小时、今天、昨天、近 3/7/14/30 天**（概览、数据分析、套餐计费共用）。

### 概览页「基础数据」板块

查询条件上方新增四个 KPI 卡片，随筛选同步更新：

| 指标 | 说明 |
|------|------|
| 总流量 | 访问总流量，自动 GB/MB 等单位 |
| EdgeOne 响应流量 | 出向流量 |
| 客户端请求流量 | 入向流量 |
| 缓存命中率 | 命中次数 ÷ 请求数 |

- **电脑端**：四个卡片横向一排  
- **手机端**：2 列网格

### 查询条件布局

- 查询按钮与时间范围、站点、域名 **同一行**（电脑端）
- 移除「筛选单个站点后可选择域名」提示文案
