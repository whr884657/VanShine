# VanShine 1.10.0 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.10.0.zip`  
**版本类型：** 大版本（规则引擎全表单化）

---

## 变更内容

### 剩余 JSON 操作全部表单化

- **访问 URL 重定向**：状态码、目标协议/域名/路径/查询参数均可下拉+输入配置
- **源站故障转移**：触发状态码 + 最多 2 条备用策略（回源/COS/S3/重定向/自定义页）
- **回源鉴权**：鉴权参数列表（URL 参数 / 请求头 + 名称 + 值）

### 未知操作通用表单

- 腾讯云控制台同步下来的、目录中未收录的操作，自动展开为嵌套表单
- 彻底移除规则编辑器中的 JSON 文本框

### SubRules ELSE 兜底分支

- 嵌套规则区新增 **「+ 添加 ELSE（兜底）」** 按钮
- ELSE 分支无需配置匹配条件，未命中上方 IF/ELSE IF 时执行
- 支持 ELSE 与 ELSE IF 混排，每个 IF 块仅允许一个 ELSE

---

- **无数据库变更**：覆盖文件即可
