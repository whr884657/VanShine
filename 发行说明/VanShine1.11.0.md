# VanShine 1.11.0 发行说明

**发行日期：** 2026-07-01  
**压缩包名称：** `VanShine1.11.0.zip`  
**版本类型：** 大版本（规则引擎移动端体验）

---

## 变更内容

### 手机端弹窗全面适配

- 条件行、嵌套规则头、操作卡片改为**单列纵向布局**，不再需要左右滑动
- 顶部**横向导航芯片**（可左右轻滑切换），点击跳转到对应 IF / EI / ELSE 块
- 底栏三按钮、输入框、下拉框均限制在屏幕宽度内

### 分支编号与条件摘要

- 外层条件块：`IF` / `IF-2`；嵌套：`嵌套IF` / `EI-1` / `EI-2` / `ELSE`
- 导航与标题使用**短标签**（路径、后缀、HOST 等），不再全部显示「HOST（域名）」
- 复合 AND 条件显示为 `HOST+路径 · xxx（2条）` 或完整「HOST 且 路径」摘要
- 列表页复合条件摘要同步修复

---

- **无数据库变更**：覆盖文件即可
