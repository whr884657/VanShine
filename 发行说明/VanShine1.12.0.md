# VanShine 1.12.0 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.12.0.zip`  
**版本类型：** 大版本（EdgeOne 缓存管理页重构）

---

## 变更内容

### 导航与命名

- 「内容与函数」→ **「缓存与函数」**
- 「内容管理」→ **「缓存管理」**（页面仍为 `content.php`，对接腾讯云 Purge/Prefetch API）

### 当前站点板块

- 手机端：站点选择框与「切换站点」按钮各占一行
- 头部新增 **刷新配额卡片**：URL / 目录 / Hostname / 全部刷新
- **付费套餐**额外显示「预热缓存」配额；**免费版**仅显示 4 项刷新配额
- 数据与「套餐计费」页 `DescribeContentQuota` 一致（今日剩余 / 日配额）

### 缓存任务表单

- 「清除缓存 / 预热缓存」切换，表单按类型动态显示
- 刷新类型联动说明与目标输入（全部刷新无需填写目标）
- 免费版隐藏预热入口并提示不支持

### 任务记录

- 任务状态中文显示（成功 / 处理中 / 失败 / 超时）
- 任务类型中文显示（URL 刷新、目录刷新等）

---

## 升级说明

- **数据库：** 无变更
- **PHP 要求：** 8.0+
- 覆盖升级后请 **Ctrl+F5** 强制刷新浏览器缓存
