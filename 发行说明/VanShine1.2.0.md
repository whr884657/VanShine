# VanShine 1.2.0 发行说明

**发行日期：** 2026-06-30  
**压缩包名称：** `VanShine1.2.0.zip`  
**版本类型：** 次版本（EdgeOne 概览功能级改版）

---

## 变更内容

### L7 访问流量多维分析

- 顶部 Tab：全部、缓存状态、国家/地区、境内省份、Host、状态码及「更多」
- 按维度展示多条折线，右侧 TOP 排行图例，支持显示/隐藏系列
- 切换 Tab 局部刷新图表（`overview_flux`），无需整页重载

### 移除访问区域分布

- 删除原环形图区域分布板块，信息合并进 L7 流量多维图

### 侧栏 KPI 卡片（6 项）

- L7 访问流量 / 带宽 / 请求数 / **防护命中次数** / 请求速率 / 缓存命中率
- 右上角标注总和、峰值或平均值
- 内联 SVG 图标 + 右下角迷你折线趋势

### 自定义筛选抽屉

- 手机端：底部抽屉（约 82% 屏高）
- 电脑端：居中弹窗，左筛选项+运算符、右侧大块选值区
- 运营商等枚举改为复选框网格，已选条件以标签展示
