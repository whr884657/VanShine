# VanShine 1.3.0 发行说明

**发行日期：** 2026-06-30  
**压缩包名称：** `VanShine1.3.0.zip`  
**版本类型：** 大版本（EdgeOne 站点管理页整页改版）

---

## 变更内容

### 站点数据概览（对标腾讯云官方）

- 左侧 4 项 KPI：总流量、总带宽峰值、总请求数、防护命中次数（含 sparkline）
- 右侧折线图 + Tab：流量 / 带宽 / 请求数 / 防护命中次数 / 缓存命中率
- 时间范围可选；支持收起/展开概览区
- 数据为账号下全部站点聚合

### 官方式站点列表

- 列：站点/ID、服务区域、接入方式、生效状态、套餐信息、标签、操作
- 搜索站点名称与 ZoneId
- 操作链接：域名管理、站点加速、安全防护（自动切换当前站点并跳转）

### 新增站点

- 「+ 新增站点」按钮 + 抽屉表单
- 仅当存在**未绑定站点**的套餐时可用；套餐下拉仅显示空槽套餐

### 本页移除

- 顶部全局「选择站点」下拉
- 站点验证信息（`describeIdentifications`）
- CNAME 配置状态（归属域名加速页）

---

## 升级说明

- **无数据库变更**：覆盖文件即可
- 自 v1.2.0 升级：概览页能力保持不变，站点管理页为本次主要变更
