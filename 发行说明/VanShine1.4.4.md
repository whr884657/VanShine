# VanShine 1.4.4 发行说明

**发行日期：** 2026-06-29  
**压缩包名称：** `VanShine1.4.4.zip`  
**版本类型：** 小版本（EdgeOne 域名管理体验优化）

---

## 变更内容

### 添加域名与列表刷新

- 添加/编辑域名成功后**关闭抽屉并局部刷新列表**，不再整页 reload，修复关闭弹窗后页面卡顿
- 添加域名只需输入**子域名前缀**（如 `www`），系统自动拼接站点已绑定的一级域名
- 列表工具栏新增「**刷新列表**」；状态为「部署中」的域名行旁有**单行刷新**按钮，用于 DNS 解析完成后更新状态

### HTTPS 证书

- 证书状态区分：**未部署 / 申请中 / 部署中 / 已部署 / 即将过期 / 已过期 / 部署失败**
- 结合 EdgeOne 默认证书 API 查询真实部署与到期时间，避免「还在申请却显示已部署」
- 部署免费证书、验证证书时**不再整页跳转**，抽屉内 toast 提示并自动轮询状态
- 证书抽屉新增「刷新证书状态」按钮
