# VanShine 1.7.0 发行说明

**发行日期：** 2026-06-30  
**压缩包名称：** `VanShine1.7.0.zip`  
**版本类型：** 大版本（规则引擎小白友好化 + 文档双仓库）

---

## 变更内容

### 规则引擎 · 面向非技术用户优化

1. **子规则（SubRules）**：与主 IF 块相同，使用下拉选择匹配类型、运算符、操作；不再默认展示 Condition 字符串与 Actions JSON
2. **常用操作表单化**：节点缓存（跟随源站/不缓存/自定义 TTL）、浏览器缓存 TTL、智能压缩等改为开关、下拉、数字输入
3. **高级表达式**：收进「开发者选项」折叠区，默认隐藏，仅技术人员需要时展开
4. **编辑器布局**：保存/取消按钮固定于弹窗**底栏**；手机端条件行改为两列网格
5. **规则列表**：手机端「规则名称 + 启用开关」保持**同一行**（名称过长省略号）
